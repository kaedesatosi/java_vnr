# java_vnr

vnr的java版小demo
- 包含剪贴板和Textractor两种提取文本方法
- 简单集成了谷歌翻译和百度翻译

翻译流程：
1. 选择读取待翻译文本的方法（剪贴板或者Textractor）
2. 选择翻译引擎
3. 如选择剪贴板方式，选中待翻译文本ctrl+c复制，软件便会进行翻译
4. 如选择Textractor方式，在弹出框中选择要翻译程序的名字和id，然后选择读取文本的方法即可
